# Task 2 - Sensor Based Simulation

## Project Title
Temperature Sensor Based LED Control System

## Description
This project is an IoT-based simulation created using Arduino UNO and a temperature sensor. The system monitors temperature values and controls an LED based on the sensor input.

## Working Principle
The temperature sensor detects surrounding temperature and sends signals to the Arduino. The Arduino processes the sensor data and controls the LED accordingly. When the temperature increases beyond a set value, the LED turns ON.

## Components Used
- Arduino UNO
- Temperature Sensor (TMP36)
- LED
- Resistor
- Jumper Wires
